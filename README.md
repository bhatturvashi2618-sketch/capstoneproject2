# Loan Default Prediction using Machine Learning

## Overview

This capstone project develops a machine learning model to predict whether a borrower is likely to default on a loan. By analyzing borrower information, loan details, and credit history, the model helps identify high-risk applicants and supports better lending decisions.

## Dataset

* **Source:** Kaggle (LendingClub Loan Dataset)
* **Target Variable:**

  * **0** = Fully Paid
  * **1** = Default (Charged Off / Default / Late Payment)

## Project Workflow

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Missing Value Treatment
* Feature Encoding
* Model Training and Evaluation
* Model Comparison
* Feature Importance Analysis

## Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest
* XGBoost

## Evaluation Metrics

Models were evaluated using:

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC Score

## Results

Among the evaluated models, **Random Forest** achieved the highest overall accuracy, while **XGBoost** provided a balanced performance after optimization. The project demonstrates how machine learning can be applied to credit risk assessment and support more informed lending decisions.
